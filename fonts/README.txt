Drop your .ttf or .otf font files into this folder.

When the application launches, it will automatically load these fonts into the UI.

RECOMMENDED FONTS TO DOWNLOAD (from Google Fonts):
1. Nunito (Specifically: Regular, Medium, SemiBold, ExtraBold, Black)
2. Quicksand (Regular / Bold)
3. Cascadia Code (Used for the Console / Terminal output)

If these fonts are not found in this folder, the application will cleanly fallback to standard Windows fonts like "Segoe UI" or "Arial", so the app will still work perfectly fine without them!
